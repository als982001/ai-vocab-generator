# Project Context

- Name: Snap-Voca (AI Japanese Vocabulary Generator)
- Goal: MVP for Frontend Portfolio. Focus on Clean Code & UX.
- Tech Stack: React (Vite), TypeScript, Tailwind CSS, Lucide React (Icons).

# Coding Rules (STRICT)

1. **Functional Components:** Use React Hooks only.
2. **TypeScript:** NO `any` type. Define interfaces in `/src/types`.
3. **Styling:** Use Tailwind CSS classes. No inline styles.
4. **Mocking:** Before implementing real API, ALWAYS use `MOCK_WORDS` from `@/mockData` to test UI.
5. **Language:** Variable names in English, UI Text in Korean.

# Directory Structure

- /src/components: UI parts (Header, UploadArea, WordCard, FloatingButton)
- /src/types: Type definitions
- /src/mockData.ts: Test data
